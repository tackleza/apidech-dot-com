MC-TH-STEVE Privacy Policy

Last updated: June 12, 2026

Overview

MC-TH-STEVE is a Thai-language Minecraft roleplay and chat assistant for Discord servers. The bot replies when users mention it in a server text channel. It is operated by Apidech Tearpaiboon.

Data the bot processes

When a user mentions MC-TH-STEVE, the bot processes the message content needed to understand the request and generate a reply. This may include the text of the message, the Discord user ID, the channel ID, the server ID, and timestamps used for rate limiting and abuse prevention.

Messages that do not mention the bot are ignored for chat response purposes.

How the data is used

Message content is used to generate an in-character reply, keep short-term conversation context, answer Minecraft questions, and prevent spam or abuse. Minecraft-related questions may be used to search a Minecraft knowledge base before the bot replies.

Storage and retention

MC-TH-STEVE does not maintain a permanent archive of user message content. The bot keeps a small amount of recent conversation context in memory so replies can follow the current channel conversation. This temporary context expires automatically.

Operational logs and rate-limit data may be kept briefly for debugging, reliability, and abuse prevention. Logs are rotated and removed after a short retention period.

Third-party processing

To generate responses, message content may be sent temporarily to the configured AI service provider. This is done only to create the bot's response. Message content from MC-TH-STEVE is not used by the operator to train machine learning or AI models.

User choice

Users can opt out by not mentioning the bot. Server administrators can restrict where the bot is available, remove the bot from a server, or contact the operator with questions about bot usage.

Children and sensitive information

Users should not send passwords, tokens, private account information, payment information, or other sensitive personal data to the bot.

Contact
